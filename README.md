# Administrationsgränssnitt - Projektuppgift - Backend-baserad Webbutveckling Mittuniversitetet

##  Syfte

Detta är en avslutande projektuppgift i kursen Backend-baserad webbutveckling vid Mittuniversitetet. Syftet med projektet är att skapa ett administrationsgränssnitt tillhörande en webbplats för ett fiktivt katt-café som använder en webbtjänst för hantering av data för meny, användaruppgifter, meddelanden samt bokningar. Webbtjänsten använder autentisering och skyddar resurser genom användning av JWT (JSON Web Token). 
På denna webbplats ska behöriga kunna logga in för att få åtkomst till skyddad data via API:et. Data lagras i en MongoDB-databas som hostas via molntjänsten MongoDB Atlas. På denna webbplats kan anävndare logga in, registrera nya användare (endast när användare är inloggade), se meny, uppdatera menyn, radera items från menyn, se meddelanden och radera meddelanden, se och radera bokningar.  



##  Installation

1. Klona repositoryt

2. Gå in i projektmappen

3. Installera npm-paket:
npm install

6. Starta servern:
npm run dev
